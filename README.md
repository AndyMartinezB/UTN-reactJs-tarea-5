# Tarea Nº5: Lista de tareas con hooks

## Descripción del Proyecto

Este proyecto corresponde a la quinta entrega del Curso de Desarrollo en React JS de
la UTN.BA. La tarea consiste en realizar una lista de tareas utilizando Hooks. El objetivo es practicar el uso de los Hooks principales como useState, useEffect para gestionar estados y efectos secundarios, useRef para manejar elementos del DOM, useCallback/useMemo para optimizar cálculos y además practicar la creación y uso de Custom Hooks.
 

## Instrucciones

Para clonar el repositorio en tu terminal:

```bash
git clone https://github.com/AndyMartinezB/UTN-reactJs-tarea-4.git
```

Para ejecutar el proyecto (React + Vite):

```bash
npm install
npm run dev
```

Luego abrir el navegador en la URL indicada por Vite (por defecto `http://localhost:5173`).
